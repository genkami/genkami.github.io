Vagrantというものを使うと、設定ファイルに記述された構成の仮想マシンを簡単に構築することができます。これを使うことで、例えば本番環境と同様の構成のテスト環境を用意したり、ということが簡単にできるようになります。

VirtualBox, Vagrantのインストール

VagrantはVirtualBoxを使って動いているようなので、とりあえず以下のサイトからVirtualBoxをインストールします。
Oracle VM VirtualBox

その後にこちらからVagrantもインストール
Vagrant by HashiCorp

VCCWを使う

今回の目的はWordPress環境をサクっと構築することなので、VagrantのWordPress環境として一番有名(っぽい)VCCWというのを使ってみます。
まずは以下のリポジトリをcloneします。
GitHub - vccw-team/vccw: A Vagrant based development environment.

$ git clone https://github.com/vccw-team/vccw

その後cloneしてきたvccwディレクトリ内で

$ vagrant up

して、あとはしばらく待つだけ。
ブラウザから http://192.168.33.10 にアクセスすれば、サーバーが動作していることが確認できます。

VCCWの便利な所は、作成した仮想マシンがホストのvccw/wwwディレクトリの中身を自動で同期してくれる所です。
適当にPHPとかを書いて適当にこのディレクトリに投げ込むだけで適当に表示できるので、とりあえず適当にPHPを書いてみたい場合には重宝します。
