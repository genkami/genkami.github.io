---

QEMUは`-hda fat:`オプションを使うことによって、ホスト上のディレクトリをFAT32のハードディスクとしてゲスト側に認識させることができます。

```
-hda fat:path/to/dir
```

デフォルトは読み込み専用なので。書き込みもしたいときは`rw`を追加します。

```
-hda fat:rw:path/to/dir
```

ちなみに、QEMUの起動後にホスト側でディレクトリに変更を加えても検知されないので注意が必要です。
